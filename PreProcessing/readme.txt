# Pre-processing codes used in Research

## combineGPR and pre-processing.R
This is the mostly used one. This will takes a folder, merged all the GPR files inside, normalized the data and output one merged file

## R merge one excel data.R
This takes in one big file and merge the same samples. This is now rarely used since IMS does not need replicates anymore.

## merging gpr in multiple folders into one.txt
This is the cmd code for merging files in multiple folders into one folder. Can be used as preprocessing step for "ombineGPR and pre-processing.R"
